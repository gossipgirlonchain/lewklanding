# Gossip Labs Landing Page

A landing page for Gossip Labs (parent company of On Cue) featuring an interactive torch effect that reveals content as you move your mouse.

## Features

- **Torch Effect**: Black overlay that reveals the underlying content in a circular area following your mouse cursor
- **Retro Pixelated Design**: Stylized text with pixelated aesthetic
- **Responsive**: Works on desktop and mobile devices

## Structure

- `index.html` - Main HTML structure
- `styles.css` - Styling and layout
- `script.js` - Torch effect JavaScript implementation

## Usage

Simply open `index.html` in a web browser. Move your mouse around to see the torch effect reveal the content underneath.

## Design Elements

- **GOSSIP LABS**: Large pink pixelated text in the center
- **ON CUE**: Smaller white text in the upper right
- **CHIPPS**: White oval logo below GOSSIP LABS
- **Portrait**: Woman's portrait in the bottom right corner
- **Black Background**: With white border frame
